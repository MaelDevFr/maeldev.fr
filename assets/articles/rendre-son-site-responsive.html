Comment rendre son site responsive ?

4m de lecture


Step 1 : Qu'est-ce que le responsive ?

• Le responsive est le terme employé pour désigner un site web s'adaptant à des tailles d'écrans différentes.
Il y a principalement, 3 tailles d'écrans à gérer : Le téléphone (480px), la tablette (780px) et les ordinateur (1280px)
Rendre son site web responsive est OBLIGATOIRE. Au minimum pour les téléphones et les ordinateurs
car vous ouvrirez votre site à une audience plus large

Step 2 : Le code :

• Pour rendre son site responsive, il vous faudra lier un fichier html à un fichier css
Ps : Je pars du principe que vous avez les bases de html & css pour ce tuto.
Et nous allons utiliser la même chose que pour le tuto du dark mode à savoir une media querie qui va fonctionner
de façon à ce que si la taille de l'écran du visiteur est inférieur ou égal à celle donnée,
alors il va lui afficher la page avec les modifications dans la media querie mais tout
en prenant en compte les éléments non modifiés qui ne sont pas dans la media querie ( pas besoin de remettre tout votre code,
juste ce que vous voulez modifier)


@media (max-width: 480px) {
html{width: 50%}
}


Et voilà, vous pouvez à présent modifier les éléments que vous souhaitez dans la media querie.
Ps : Cet exemple vous donnera un site responsive pour téléphone.

Ps : Et bootstrap dans tout ça ?

Ah ! Voilà une question qui vous intéresse peut être. Vous avez peut être
déjà entendus parler de "bootstrap" comme "le framework css qui rend ton site responsive" Et bien ne l'utilisez pas !
Pourquoi ?
Car avec un outil comme bootstrap vous êtes bien trop dépendant de l'outil. Certes il peut être pratique
des des cas très spécifiques mais si votre but est de créer un bon site internet
avec un beau design, bootstrap n'est pas la solution et privilégiez du CSS pure. Mais si
vous voulez tout de même utiliser un framework css, alors tailwindCss vous conviendra mieux.
