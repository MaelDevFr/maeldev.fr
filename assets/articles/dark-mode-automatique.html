Comment créer un dark-mode automatique en CSS ?

3m de lecture
Step 1 : Un fichier CSS :
• Pour avoir votre dark-mode automatique, vous devez lier un fichier css à un fichier HTML Ps : A noter que je pars du principe que vous possédez les bases d'Html & Css pour ce tuto. Vous devez insérer dans le fichier CSS le code suivant : 

              
                  @media (prefers-color-scheme: dark) {
                     html{
                       background-color: black;
                     }
                  }


• Et paf ! Si votre ordinateur à pour préférence le dark mode,
le fond de votre page deviendra noire ! C'est pas magique juste du CSS !
Step 2 : Aller plus loin :
• Évidemment vous pouvez aller plus loin avec cette option, vous pouvez changer la couleur du texte, des animations en séléctionnant les éléments html que vous voulez changer à l'intérieur de la media queries et les changer.
