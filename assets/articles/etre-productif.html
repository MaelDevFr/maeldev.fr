Comment être productif lors de ses sessions de code ?

2m de lecture


Step 1 : Etre productif

• Vous pouvez vous poser la question : comment être productif lors de sessions de code ?
Sur cette article je vais vous partagez plusieurs " astuces " pour devenir productif.
Bien sûr ces techniques ne marchent pas forcément pour tout le monde, elles se
basent sur mon expérience réduite mais je pense qu'elle pourrait vous aider aussi.
Voici donc différents thèmes de la productivité.

Step 2 : L'ambiance :

• Pour être productif, il est bien connus une ambiance sonore est indispensable
Pour les différents types de musiques, c'est entièrement personel en fonction des personnes
et je ne pourrais pas choisir pour vous, il faut que vous testez les différents types de musiques.
Mais voici les types de musiques les plus utilisés :

• Jazz
• Lofi

Step 3 : Le thème de l'ordinateur :

• Bien sûr le thème de l'ordinateur est primordial. Il y a 2 thèmes :

• Le Dark Mode ( mode sombre )
• Le Light Mode ( mode clair )

Pour la santé de vos yeux je ne saurais que vous conseillez le dark mode car le
light mode est beaucoup trop clair notemment pour la coloration syntaxique du code
et si vous codez la nuit avec le light mode je vous souhaite bien du courage !.
Vous l'aurez compris des 2 le dark mode est le mieux ( encore une fois un avis subjectif mais
que beaucoup d'autres partagent )

Step 4 : Et vous :

• Et bien évidemment, vous êtes le plus important des facteurs de la productivité.
Vous devez pour être productif sur un projet, vous concentrez à 100 % sur ce projet.
Vous dire que tous les mardis à 15h vous prenez 1h à ne faire qu'avancer sur votre projet.
Il est important de se fixer des rituels de session de code afin de ne pas perdre la motivation.
