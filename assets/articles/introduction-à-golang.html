Introduction à GoLang

12m de lecture
Step 1 : Qu'est-ce que GoLang ?
• GoLang est un langage de programmation créer par Google qui à été lancé pour la première fois en 2009. C'est un langage compilé dont la syntaxe est très ressemblante à celle de Python et de C. Le langage est écrit en C et est considéré comme un langage 'bas-niveau' ( proche du processeur ). Si vous avez observé mon site, vous devez savoir que j'adore ce langage pour une bonne raison : On peut tout faire en Go ! Applications web, apps desktop, tâches systèmes etc... tout ça très simplement. En effet, le GoLang à pour slogan d'être un langage de programmation "beginner-friendly". En effet le GoLang n'a que 25 mots clés là ou JavaScript en à 64. De plus, Go à développé son écosystème avec une masse de packages & frameworks au point où, aujourd'hui, n'importe quelle tâche peut être réalisée en Go.
Step 2 : Comment on l'installe :
• Franchement, de ce côté là, Go à réalisé la perfection. Vous n'avez qu'un package à installer sur votre machine, quelques variables à définir et... c'est tout ! Pour installer Go rendez vous ici et télécharger la version de go adaptée pour votre OS. Une fois que ceci est fait, vous devez avoir à la racine de votre disque dur un dossier nommé "go". Ouvrez ce dossier et créer un dossier et appelez-le "src", ce sera là-dedans que vous stockerez vos projets GO. Ensuite, définissez une variable d'environnement et nommez-là GOPATH et définissez-là dans le dossier "go".
Pour définir une variable d'environnement sous windows :

GOPATH=votreChemin


Et pour définir sous macOS ou linux :

export GOPATH=votreChemin


Une fois que c'est fait, vous pouvez si vous le souhaitez ( je vous le recommande ) installer l'extension VSCode "Go" pour avoir une meilleur confort de coding.
Et voilà ! C'est fait, c'était rapide n'est-ce pas ? Sachez que ce n'est que le début !
Step 3 : Hello World : 

            
                package main
                

                import "fmt"
                

                func main() {
                  fmt.println("Hello World !")
                }

• Examinons à présent ce code. On va décomposer ce code en plusieurs morceaux.
1) Le "package main", cette ligne permets de définir dans quel package le fichier s'inscrit. Cela vous sera utile lorsque vous décomposerez vos gros fichiers en plusieurs petits.
2) Le "import "fmt"", cette ligne permets d'importer un module Go. C'est très important car sans ce module, vous ne pouvez pas print dans la console.
3) La fonction main. Cette fonction est très importante car c'est elle qui va s'éxécuter, elle et aucune autre variable. Vous devez mettre dans la fonction main les fonctions externes que vous voulez utiliser. Chaque projet ne doit avoir en théorie qu'une seule fonction main.
Step 4 : La compilation :
• Et voilà, après une journée de dur labeur, vous avez enfin fini votre code Go. Maintenant, il faut le mettre en production et le compiler. Sur ce point là, encore une fois, golang nous surprends par sa rapidité et sa facilité de compilation cross-plateform. Ce que Go fait lorsque vous compilé un programme, c'est qu'il va créer un fichier binnaire pour l'os que vous avez demandé ou si vous n'avez pas précisé quel os, il va compiler pour votre os. Bon, c'est bien beau mais, comment on fait ? Nous allons partir du principe que vous voulez compilé pour votre os. En réalité c'est très très simple, rentrez simplement dans votre terminal cette commande :

go build main.go


Et voilà, quelques secondes plus tard, vous avez un fichier que vous pouvez exécuter qui s'est créer ! C'est pas de la magie, juste GoLang.
Step 5 : Conclusion :
• En bref, je pense que vous l'avez deviné mais GoLang est, selon moi, l'un des meilleurs langages de programmation disponible sur le marché. Il peut clairement concurrencer JavaScript si seulement il était si connus que ce dernier. Mais ne vous découragez pas ! Devenez un Gopher et agrandissez la communauté golang !
