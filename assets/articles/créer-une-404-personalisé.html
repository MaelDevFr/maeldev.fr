Comment créer une page d'erreur 404 personalisé ?

2m de lecture
Step 1 : Un fichier spécial :
• Pour avoir une page d'erreur personalisée il vous faut un fichier spécial dans votre code Ps : A noter que je pars du principe que vous possédez un serveur apache pour ce tuto. Ce fichier doit être nommé .htaccess et vous devez insérer dedans le code suivant :

ErrorDocument 404 /erreur404.html


Vous devez également créer un fichier à la racine de votre projet nommé
erreur404.html
Step 2 : Le contenu :
• Pour faire une bonne page d'erreur 404, vous devez mettre l'utilisateur sur la bonne voie, avec maximum 2 calls to action. L'un pour revenir sur la page d'acueil et l'autre si besoin vers la partie contact du site si une erreur c'est produite
